Mighty Math Markdown

Mighty Math Markdown is:

  • A syntax that lets you type math the way you already know how.
  • A library that renders that syntax into beautiful HTML math (MathML) on your webpage.

Mighty Math Markdown is based on the way people already write math using plain ASCII text in emails and on web forums.

Mighty Math Markdown is named after John Gruber’s original Markdown. It’s called "Mighty" to distinguish it from any other products that might be called Math Markdown, and because I like alliteration.

Examples

x = -b +/- sqrt(b^2 - 4.a.c) / 2a

gives

%% x = -b +/- sqrt(b^2 - 4.a.c) / 2a %%
